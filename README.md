# Trip-Avishi

# A full-stack trip planner web app built using the **MERN stack** where users can register, log in, and plan multiple trips with detailed itineraries.


## 📁 Project Structure

Trip Planner/
├── frontend/        # React frontend
├── backend/         # Node.js Express backend
└── README.md


## 🚀 How to Run

### 🔹 1. Clone the Repo

```bash
git clone https://github.com/ecloudops-tech/Trip-Avishi.git
cd Trip-Avishi
```

---

### 🔹 2. Run Backend

```bash
cd backend
npm install
npm start
```

> Server will run.

---

### 🔹 3. Run Frontend
```bash
cd frontend
npm start
```

---
